# 区块链技术简介

## 什么是区块链？

区块链是一种分布式账本技术，它将数据存储在称为"区块"的数据结构中，并通过密码学方法将多个区块链接成链式结构。

## 核心特性

### 去中心化
传统数据库由单一机构控制，而区块链数据存储在网络中的多个节点上，没有单一控制点。

### 不可篡改
每个区块都包含前一个区块的哈希值，任何对历史数据的篡改都会导致哈希值不匹配，从而被发现。

### 可追溯
每笔交易都被记录在区块中，并且可以追溯到最初的发起者。

## 共识机制

| 机制 | 原理 | 代表应用 |
|------|------|---------|
| PoW | 工作量证明，消耗算力 | 比特币 |
| PoS | 权益证明，持有量越多权利越大 | 以太坊2.0 |
| DPoS | 委托权益证明，选举代表验证 | EOS |

## 典型应用场景

1. **数字货币**：比特币、以太坊等加密货币
2. **智能合约**：自动执行的数字化协议
3. **供应链追溯**：商品全流程可查证
4. **数字身份**：个人信息的去中心化身份认证
